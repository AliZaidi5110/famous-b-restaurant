# Famous B Restaurant Website

A professional, mobile-responsive website for Famous B Restaurant built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **Image Slider**: Auto-rotating hero images every 6 seconds
- **Video Section**: Embedded restaurant video
- **Menu Section**: PDF menu download
- **Contact Section**: Location, phone, email, and opening hours
- **Mobile Responsive**: Fully responsive design for all devices
- **SEO Optimized**: Meta tags and keywords for search engine optimization

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build for Production

```bash
npm run build
npm start
```
